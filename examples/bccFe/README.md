# bcc Fe

Monte Carlo examples for body-centered cubic (bcc) iron using a Spin-Cluster Expansion (SCE) model.

## System

The SCE model was constructed from a **2×2×2 supercell of the cubic bcc unit cell** (16 Fe atoms, lattice parameter 2.8298 Å),
by sampling spin configurations from a **Curie-Weiss paramagnetic state** (fully disordered magnetic state).
The model includes **2-body clusters up to 2nd nearest neighbors with lsum = 2** (Heisenberg-type interactions).
The spin–spin interactions are encoded in `jphi.xml` generated by [Magesty.jl](https://github.com/cometscome/Magesty.jl).

## Examples
